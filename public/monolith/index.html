Installation

To start your project with Lucid installed, run the following command:

composer create-project lucid-arch/laravel my-project

This will download and install the latest Laravel version with Lucid out-of-the-box.

It ships with the Lucid Console which provides a command line interface that is useful for scaffolding and navigating Services, Features, and Jobs.

Install Custom Version

If you wish to download other versions of Laravel you may specify it as well:

composer create-project lucid-arch/laravel=7.x my-project-7.x

This would install the latest in Laravel v7.

Setup

The lucid executable will be in vendor/bin. If you don’t have ./vendor/bin/ as part of your PATH you will need to execute it using ./vendor/bin/lucid, otherwise add it with the following command to be able to simply call lucid:

export PATH="./vendor/bin:$PATH"

For a list of all the commands that are available run lucid or see the CLI Reference.
